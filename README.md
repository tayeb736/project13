# Cycode Enterprise Website 🚀

Modern, high-performance corporate website for Cycode, an enterprise-grade software development company.

## ✨ Features
- **Modern UI/UX**: Built with premium aesthetics and smooth animations.
- **Responsive Design**: Optimized for Desktop, Tablet, and Mobile.
- **Interactive Playground**: Real-time code editor and preview.
- **Multilingual Support**: Integration with Google Translate for global reach.
- **Dark/Light Mode**: Seamless theme switching with persistent settings.
- **Advanced Security**: Built with secure coding standards.

## 🛠️ Tech Stack
- **Structure**: HTML5, Semantic UI
- **Styling**: Vanilla CSS3, Grid, Flexbox
- **Logic**: Modern JavaScript (ES6+)
- **Icons**: SVG, Simple Icons, FontAwesome
- **Deployment**: GitHub Pages

## 🚀 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/tayeb736/project13.git
   ```
2. Open `index.html` in your browser.
