# Cycode Enterprise Website 🚀

Modern, high-performance corporate website for Cycode, an enterprise-grade software development company.

## ✨ Features
- **Native Multi-Language**: Built-in support for English, French, and Arabic without external dependencies.
- **Modern UI/UX**: Premium aesthetics and smooth animations.
- **Mobile Ready**: Fully configured as a cross-platform mobile app using Capacitor.
- **Interactive Playground**: Real-time code editor and preview.

## 🛠️ Tech Stack
- **Web**: HTML5, CSS3, JavaScript (ES6+)
- **Mobile**: Capacitor, Android
- **Hosting**: GitHub Pages

## 🚀 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/tayeb736/project13.git
   ```
2. For web: Open `index.html` in your browser.
3. For mobile: Open the `android` folder in Android Studio.
